申请GitHub

  1. 进入GitHub注册账号；
  2. 注册成功后，新建repository；
  3. 输入Repository name:yourname.github.io(yourname与你的注册用户名一致，这就是博客地址）；
  4. Repository建立完成，进入Setting–>Options，GitHub Pages模块，点击“Launch automatic page generator”，启动GitHub Page；
  5. 点击“Continue to layouts”，最后点击“Publish page”,发布github默认生成的一个静态站点。

常见问题

  • Q: 摘要怎么在首页中就剩下140个字
    A: 为了提高打开首页的速度故首页不加载文章图片，只加载文章摘要。强烈建议大家在写文章的时候自己总结description并将其放在开头的front-matter中。如果没有会截取内容前的140个字。
  • Q: 审美不同，想换个颜色怎么办？
    A: 在variable.styl文件中修改。
  • Q: pacman主题语法不能高亮
    A：在pacman中的_config.yml后面加入
    highlight:
enable: true
  • Q: hexo deploy时重复输入用户名密码的问题
    A: 最主要的原因可能是木有采用git@github.com，而是用https//github.com，修改后重新部署即可。
  • Q: hexo deploy不能用
    A: _config.yml文件中找到deploy:标签，在每个冒号后面必须要空格。
