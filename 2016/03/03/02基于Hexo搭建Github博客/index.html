申请GitHub

  1. 进入GitHub注册账号；
  2. 注册成功后，新建repository；
  3. 输入Repository name:yourname.github.io(yourname与你的注册用户名一致，这就是博客地址）；
  4. Repository建立完成，进入Setting–>Options，GitHub Pages模块，点击“Launch automatic page generator”，启动GitHub Page；
  5. 点击“Continue to layouts”，最后点击“Publish page”,发布github默认生成的一个静态站点。
